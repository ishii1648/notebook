# Notebook リポジトリ

学習した内容を分野ごとに分類してまとめるリポジトリ。

## ディレクトリ構造

- 分野ごとにトップレベルディレクトリを作成する（例: `coding-agent/`, `k8s/`, `service-mesh/`）
- 各分野ディレクトリの下に、読んだ記事や学習ノートを配置する
- 各分野ディレクトリには `INDEX.md` を置き、記事リストとして一覧できるようにする

```
<分野名>/
  INDEX.md        # 記事リスト（タイトル・URL・要約など）
  <記事ノート>.md  # 個別の記事メモ
```

## 運用ルール

- 新しい分野を追加する場合はトップレベルにディレクトリを作成し、`INDEX.md` を必ず作成する
- 記事を追加したら対応する `INDEX.md` も更新する
